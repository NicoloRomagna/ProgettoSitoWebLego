# Progetto Sito Web Lego

## Descrizione del Progetto
Il progetto ha una funzione di register e login e inoltre riguarda la creazione di un sito web dedicato al mondo dei Lego. 
Il sito presenterà una vasta gamma di prodotti Lego, insieme alla sua storia interessante un catalogo per acquistare i singoli pezzi.

### Contenuti Aggiuntivi
Il sito includerà anche immagini, articoli e video che mostrano come costruire diversi modelli Lego.
